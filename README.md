# Project Setup Instructions

This project is a simple personal portfolio website built with **HTML**, **CSS**, and **vanilla JavaScript**.  
No external libraries or build tools are required.

---

## Requirements

To run this project you only need:
- A modern web browser (Chrome, Firefox, Edge, or Safari)
- A code editor (for example Visual Studio Code)

---

## Project Structure

project-folder/
│
├── index.html
├── style.css
├── script.js
├── README.md
└── images/


- `index.html` – Main HTML file  
- `style.css` – Styling for the website  
- `script.js` – JavaScript for the contact form validation  
- `images/` – Folder containing images used on the website  

---

## How to Run the Project

1. Download or clone the project repository.
2. Open the project folder on your computer.
3. Open the file `index.html` in a web browser  
   (double-click the file or right-click and choose **Open with browser**).

The website will load locally in your browser.

---

## How to Edit the Project

1. Open the project folder in your code editor.
2. Edit:
   - `index.html` to change content and structure
   - `style.css` to change design and layout
   - `script.js` to update form validation logic
3. Save the files and refresh the browser to see changes.

---

## Contact Form Functionality

The contact form uses JavaScript to:
- Validate required fields
- Validate email format
- Show error messages
- Display a success message when the form is submitted
- Count characters in the message field

The form does not send real emails. It is intended for demonstration and learning purposes.

---

## Notes

- No installation or configuration is required.
- No backend or database is used.
- The project runs entirely in the browser.
